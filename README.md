# Unity3DGameBuilds
Unity3D小游戏

### Space_Shooter  飞行射击游戏
使用Unity3D和C#语言进行开发。玩家可以控制飞机进行移动和射击，敌方飞机将分波次进行下落进攻，敌机可以左右随机闪避，并且瞄准玩家飞机位置射击。<br>
